About This Code - Memory-Spiel

Funktionsweise: Das Popup-Fenster ermöglicht es dem Benutzer, zwischen "Fruits Picture" und "Hardware Picture" zu wählen, um die Art der Bilder auszuwählen, die im Memory-Spiel verwendet werden sollen. Nach Auswahl der Bilder muss das Fenster mit dem X wieder geschlossen werden. Das Spiel besteht aus einem Spielfeld mit einer Anzahl von Karten, die zu Beginn verdeckt sind. Der Spieler kann zwei Karten durch Anklicken auswählen, um sie aufzudecken. Wenn die aufgedeckten Karten dasselbe Bild haben, bleiben sie offen, und der Spieler erhält einen Punkt und darf weitere Karten aufdecken. Wenn die Karten unterschiedliche Bilder haben, werden sie wieder umgedreht, und der Spieler kann zwei neue Karten auswählen. Das Spiel endet, wenn alle Paare gefunden wurden, und der Spieler erhält eine Benachrichtigung über seinen Erfolg. Code-Struktur: index_structure.html: Die Haupt-HTML-Datei, die das Spielfeld und das Popup-Fenster für die Bildauswahl enthält. picture_selection.js: JavaScript-Datei, die die Funktionen zur Auswahl der Bilder enthält. Hier kann der Spieler zwischen "Fruits Picture" und "Hardware Picture" wählen. memory_script.js: JavaScript-Datei, die die Spiellogik und Interaktionen auf dem Spielfeld steuert. Hier werden die Karten aufgedeckt, die Paare überprüft und die Punkte berechnet. memory_cards.js: JavaScript-Datei, die das Array der Kartenobjekte enthält, einschließlich der Bildpfade und der Zuordnung zu den jeweiligen Paaren..
